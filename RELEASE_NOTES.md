## ✨ 新增功能

### 🛡️ 1. 四大究极防护体系

#### 协议层究极体
- **反主动探测（Anti-Probing）**
  - 幻影无响应模式 - 对未授权连接完全静默
  - 握手暗号验证 - SHA256 + 时间戳双重验证
  - IP 白名单机制 - 精确控制访问权限
  - 严格模式 - 非白名单直接拒绝

- **TLS 指纹伪装（Parrot Mode）**
  - 6 个预定义真实指纹（Chrome、Firefox、Safari、原神等）
  - 完整 JA3/JA4 指纹复刻
  - ALPN 协议协商伪装
  - 密码套件组合伪装

#### 防御究极体
- **反调试检测** - 支持 Windows/Linux/macOS 多平台
- **内存蜜罐** - 静默监控扫描工具（CheatEngine、ProcessHacker 等）
- **配置欺骗** - 假配置文件 + 真配置 AES-256-GCM 加密
- **自毁机制** - 检测威胁立即清除敏感数据

#### 架构层究极体
- **XDP 零内核态切换代理**（Linux）
  - 延迟降低 10 倍（100μs → 10μs）
  - 吞吐量提升 10 倍（5 Gbps → 50+ Gbps）
  - CPU 占用降低 80%
  - 网卡驱动层数据包拦截
  - 零内存拷贝

#### 路由究极体
- **多路径阴影路由**
  - 数据流分片传输，降维打击行为分析
  - 5 种分片策略（轮询、随机、加权、最少连接、延迟优先）
  - 节点池管理（通用、流媒体、游戏、下载、社交）
  - 会话绑定规则（避免 IP 乱跳导致封号）
  - 预定义安全规则（Netflix、YouTube、Steam 等必须单节点）

### 🌐 2. DNS 零泄漏防护

#### 4 种防护级别
- **无防护** - 本地开发环境
- **基础防护** ⭐ 推荐 - 强制使用 DoH，平衡速度和安全
- **严格防护** - Fake-IP 模式，阻止明文 DNS
- **偏执防护** - 最强防护，禁用 IPv6，阻止所有明文查询

#### 核心特性
- ✅ 100% DNS 查询加密（DoH/DoT）
- ✅ 防止 DNS 污染和劫持
- ✅ 防止 ISP 监控
- ✅ 内置 DNS 泄漏测试
- ✅ 智能分流（国内/国外 DNS）

## 📦 使用指南

### 快速开始

#### 1. 启用安全监控
```
设置 → 安全 → 启动安全监控
```

#### 2. 配置反探测
```
设置 → 安全 → 反主动探测
- 启用反主动探测
- 生成握手暗号
- 添加白名单 IP（可选）
```

#### 3. 选择 TLS 指纹
```
设置 → 安全 → TLS 指纹伪装
- 选择预定义指纹（推荐：Chrome 120）
```

#### 4. 启用 DNS 零泄漏防护
```
设置 → DNS → DNS 零泄漏防护
- 选择防护级别（推荐：基础防护）
- 点击"开始测试"验证配置
```

#### 5. 配置多路径路由
```
设置 → 高级 → 多路径阴影路由
- 启用多路径路由
- 选择分片策略（推荐：加权）
- 配置节点池
- 遵守预定义规则（流媒体、游戏必须单节点）
```

#### 6. 查看性能监控
```
首页 → 性能监控
- 查看实时流量、延迟、连接数
- 查看性能趋势图
- 应用优化建议
```
