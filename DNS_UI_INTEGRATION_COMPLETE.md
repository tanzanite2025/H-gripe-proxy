## 用户交互流程

### DNS 分流配置流程

1. 用户打开设置页面
2. 在 **DNS 智能分流** 卡片中选择模式
3. 系统自动应用配置
4. **DNS 统计** 卡片实时显示当前配置
5. 用户可以在 **DNS 统计** 中查看分流效果

### Tor 配置流程

1. 用户打开设置页面
2. 在 **Tor 代理** 卡片中启用 Tor
3. 配置 SOCKS5 主机和端口（可选）
4. 点击 **检查连接** 验证 Tor 状态
5. **DNS 统计** 卡片显示 Tor 状态
6. 用户可以复制 SOCKS5 代理地址用于其他应用
