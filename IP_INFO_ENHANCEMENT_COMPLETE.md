## 📚 使用指南

### 基本使用

#### 1. 查看 IP 信息
```
打开应用 → 首页 → IP 信息卡片
- 自动显示当前 IP 和位置
- 每 5 分钟自动刷新
- 点击刷新按钮手动刷新
```

#### 2. 检测代理
```
打开应用 → 首页 → 代理检测卡片
- 首次使用：关闭代理 → 点击"保存为直连 IP"
- 启用代理后：自动对比检测
- 查看建议：了解代理状态
```

#### 3. 检测 DNS 泄漏
```
打开应用 → 首页 → DNS 安全检测卡片
- 点击刷新按钮开始检测
- 查看风险等级
- 查看详情：了解 DNS 服务器信息
- 按建议修复
```

#### 4. 测试网络速度
```
打开应用 → 首页 → 网络速度测试卡片
- 点击"开始测试"
- 等待测试完成（约 30-60 秒）
- 查看结果：下载/上传/延迟/丢包
- 可随时点击"停止测试"
```

#### 5. 检测 WebRTC 泄漏
```
打开应用 → 首页 → WebRTC 泄漏检测卡片
- 点击刷新按钮开始检测
- 查看风险等级
- 查看详情：了解检测到的 IP
- 按建议修复（禁用 WebRTC 或安装扩展）
```
