## ⚠️ 系统托盘图标说明

### Windows 系统托盘图标大小

Windows 系统托盘图标的显示大小**由操作系统决定**，不能通过代码直接控制：

- **标准 DPI (100%)**：16×16 像素
- **高 DPI (125%)**：20×20 像素
- **高 DPI (150%)**：24×24 像素
- **高 DPI (200%)**：32×32 像素

### 当前托盘图标配置

**ICO 文件位置：**
- `src-tauri/icons/tray-icon.ico` - 普通状态
- `src-tauri/icons/tray-icon-sys.ico` - 系统代理状态
- `src-tauri/icons/tray-icon-tun.ico` - TUN 状态

**当前 ICO 文件大小：** 54.77 KB (56,083 bytes)

**ICO 文件包含的尺寸：**
ICO 文件是多尺寸容器，通常包含：
- 16×16 - 标准托盘图标
- 24×24 - 中等 DPI
- 32×32 - 高 DPI
- 48×48 - 大图标
- 64×64 - 超高 DPI
- 256×256 - 高清预览

Windows 会根据当前 DPI 设置自动选择最合适的尺寸。

## 🎨 如何让托盘图标看起来更大/更清晰

### 方案 1：优化图标设计（推荐）

由于托盘图标尺寸受系统限制，最好的方法是优化图标设计：

1. **简化设计**：在小尺寸下使用更简单的图形
2. **增加对比度**：使用高对比度颜色
3. **加粗线条**：在 16×16 和 32×32 尺寸中使用更粗的线条
4. **去除细节**：移除在小尺寸下看不清的细节

### 方案 2：使用不同的小尺寸设计

为 ICO 文件的不同尺寸设计不同的图标版本：

- **16×16 / 32×32**：使用简化版本（更粗、更简单）
- **48×48 及以上**：使用完整版本（更多细节）

这需要专业的图标编辑工具，如：
- **IcoFX** (Windows)
- **Greenfish Icon Editor Pro** (免费)
- **Adobe Photoshop** (带 ICO 插件)

### 方案 3：调整系统 DPI 设置（用户端）

用户可以通过调整 Windows 显示缩放来让托盘图标显示更大：

1. 右键桌面 → 显示设置
2. 缩放与布局 → 更改文本、应用等项目的大小
3. 选择 125% 或 150%

**注意：** 这会影响整个系统的显示大小。
