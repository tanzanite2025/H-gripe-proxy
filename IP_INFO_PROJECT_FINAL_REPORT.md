## 📱 使用指南

### 方式 1: 首页卡片（日常使用）

1. **启用卡片**
   - 打开首页
   - 点击右上角设置图标（⚙️）
   - 勾选需要的卡片：
     - ☐ 代理检测
     - ☐ DNS 泄漏检测
     - ☐ 速度测试
     - ☐ WebRTC 泄漏检测
   - 点击保存

2. **使用卡片**
   - 卡片会显示在首页
   - 点击刷新按钮进行检测
   - 查看结果和建议

### 方式 2: 网络诊断页面（深度诊断）

1. **访问页面**
   - 点击导航栏中的"网络诊断"标签
   - 或直接访问 `/network-diagnostic`

2. **运行诊断**
   - 所有检测卡片集中显示
   - 可同时运行多个检测
   - 查看详细的诊断结果

### 推荐工作流

```
日常使用:
  首页 → 启用 IP 信息卡片 → 快速查看

深度诊断:
  网络诊断页面 → 运行所有检测 → 查看详细结果

问题排查:
  网络诊断页面 → 识别问题 → 应用修复建议
```

- 用途：测试代理节点的延迟

- 用途：测试当前网络的延迟
